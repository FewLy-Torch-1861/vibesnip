A terminal-inspired, keyboard-centric code snippet manager.


## Tech Stack
*   **Backend:** Python (FastAPI)
*   **Frontend:** HTML + HTMX + TailwindCSS
*   **Database:** SQLite

## Setup & Run

1.  **Install Dependencies:**
    ```bash
    cd VibeSnip
    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    ```

2.  **Run the App:**
    ```bash
    uvicorn main:app --reload
    ```

3.  **Open in Browser:**
    Go to `http://127.0.0.1:8000`

## Features
*   **Add Snippets:** Store code, language, and tags.
*   **Search:** Instant filtering via HTMX.
*   **Copy:** One-click copy to clipboard.
*   **Delete:** Remove old snippets instantly.
